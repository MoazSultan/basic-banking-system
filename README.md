# Basic Banking System in Python

## Overview

This is a simple console-based Banking System developed in Python using functions and dictionaries. The project simulates basic banking operations such as creating accounts, depositing money, withdrawing money, checking balances, and updating account holder names.

## Features

* Create a new bank account
* Automatically generate account numbers
* Deposit money into an account
* Withdraw money with balance validation
* Check account balance
* Change account holder name
* Menu-driven console interface

## Technologies Used

* Python
* Functions
* Dictionaries
* Conditional Statements
* Loops

## Project Structure

* `create_account()` – Creates a new account.
* `deposit()` – Adds money to an account.
* `withdraw()` – Withdraws money if sufficient balance is available.
* `check_balance()` – Displays the current account balance.
* `change_account_name()` – Updates the account holder's name.
* `generate_account_number()` – Generates unique account numbers.
* `main()` – Controls the menu-driven system.
